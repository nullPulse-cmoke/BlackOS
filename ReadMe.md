BlackOS
BlackOS is a macOS-style desktop environment that runs entirely in your browser. I built it using pure HTML, CSS, and vanilla JavaScript—no heavy frameworks involved.

Features
Interface: macOS-style dock with zoom effects, top menu bar, and glassmorphism window designs.

Core Apps: Includes a file manager (Finder), a functional browser, a terminal with custom commands, and a system settings panel.

Functionality: Drag-and-drop windows, dynamic wallpaper switching via API, and real-time system status tracking.

How to use
Just open index.html in any modern browser to get started.

Roadmap
I’m planning to add a text editor, a notification center, and a dark mode soon. Feel free to check out the code, and if you like it, a star on GitHub would be awesome!
